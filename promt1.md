# Промпт для агента: Лендинг Exclusion

## Контекст проекта

Ты создаёшь лендинг для бренда **Exclusion** — съёмные аксессуары для уличной одежды. Ключевой продукт — модульный пуллер для молний, состоящий из двух частей: постоянная основа на молнии + съёмная подвеска с кастомным дизайном. Слоган: *"Modular zipper system. One base — infinite styles."*

Главная цель сайта — **сбор заявок на предзаказ**.

---

## Технический стек

- Чистый **HTML + CSS + vanilla JavaScript**. Никаких фреймворков, никакого React, никакого TypeScript.
- Шрифт: **Manrope** (подключить через Google Fonts)
- Библиотека физики: **Matter.js** (подключить через CDN) — только для секции каталога
- Никаких других внешних библиотек

## Структура папок

```
/site
  index.html
  /css
    styles.css
    animations.css
  /js

    Render.jpg         ← фото товара (используется для всех 4 карточек)

```

> Все пути к изображениям прописывай относительно index.html. Если файла нет — используй заглушку через CSS background-color, не ломай вёрстку.

## Дизайн-система

**Цвета:**
- `--black: #0A0A0A`
- `--white: #FFFFFF`
- `--grey-light: #F5F5F5`
- `--grey-mid: #888888`
- `--blueprint-blue: #1A6BFF`

**Типографика (Manrope):**
- Display: 72-96px, weight 800, letter-spacing -0.02em
- H2: 48px, weight 700
- H3: 24px, weight 600, letter-spacing 0.1em, uppercase
- Body: 16px, weight 400, line-height 1.6
- Caption: 12px, weight 500, letter-spacing 0.15em, uppercase, color `--grey-mid`

**Сетка:**
- Максимальная ширина контента: `calc(100vw - 80px)` (по 40px с каждой стороны)
- На мобайле: `calc(100vw - 32px)`

**Радиусы:** 0px везде. Никаких скруглений.

## Общие требования

- Сайт **одностраничный**, полностью адаптивный (mobile-first)
- Анимации через **Intersection Observer** — элементы появляются при скролле (fade-in + translateY)
- Кастомный курсор: тонкий крест `+` 20x20px, цвет белый на тёмных секциях, чёрный на светлых
- Плавный скролл: `scroll-behavior: smooth`
- Переключение языка меняет весь текстовый контент через JS (data-атрибуты на элементах)

**Заголовок секции:**
- `Logo.png` (чёрная версия, 32px) + текст **PULLERS** (H3) рядом

**Карточки товаров (4 штуки, сетка 4 колонки → 2 на планшете → 1 на мобайле):**

Товары:
1. Exclusion Zipper Base — $25 / 2500₽ / 82Br
2. Exclusion EX Zipper #1 — $25 / 2500₽ / 82Br
3. Exclusion EX Zipper #2 — $25 / 2500₽ / 82Br
4. Exclusion EX Zipper #3 — $25 / 2500₽ / 82Br

Каждая карточка:
- Фон изображения: `--grey-light`
- Изображение: `Render.jpg`, object-fit: cover
- При наведении на изображение: запускается слайдер (пока все кадры = `Render.jpg`, интервал 600ms)
- Под изображением: название товара (H3, чёрный) + цена (Body, `--grey-mid`)
- Кнопка *"Предзаказ"* появляется при hover на карточку (fade-in снизу)

**Физика маятника (Matter.js):**
- В заголовке секции, перед словом PULLERS, вместо статичного `Logo.png` — физический объект
- Иконка логотипа подвешена на "цепочке" (нарисованной через Canvas или SVG) к верхней границе секции
- При скролле вниз/вверх — маятник получает импульс и раскачивается с затуханием
- Реализовать через Matter.js: тело = прямоугольник с изображением Logo.png, верхняя точка крепления фиксирована
- Движение мышью по секции тоже даёт лёгкий импульс

### 3. Blueprint — система крепления

**Это scroll-locked секция. Логика:**
1. Когда пользователь доскролливает до секции — скролл страницы блокируется (`overflow: hidden` на body)
2. Внутри секции проигрывается анимация: SVG чертёж пуллера "рисуется" (stroke-dashoffset от длины пути до 0)
3. После завершения анимации чертежа — появляются (fade-in) 3 панели с пояснениями
4. Только после того как все панели появились — скролл страницы разблокируется
5. Пользователь может скроллить дальше

**Визуал:**
- Фон: `--blueprint-blue` (#1A6BFF)
- Сетка чертежа: тонкие белые линии opacity 0.1 (`background-size: 40px 40px`)
- SVG чертёж: схематичное изображение пуллера (две части — основа и съёмная подвеска) в стиле технического чертежа. Белые линии на синем фоне. Нарисуй SVG сам геометрическими примитивами.
- Анимация: `stroke-dasharray` + `stroke-dashoffset` через CSS animation, запускается через JS когда секция входит во viewport

**Три панели с пояснениями (появляются последовательно после чертежа):**
1. *"Базовая часть"* — крепится на молнию навсегда, служит фиксатором
2. *"Съёмная подвеска"* — кастомный дизайн, меняется нажатием
3. *"Система фиксации"* — пластиковый крепёж: вставил и зафиксировал, снял нажатием

Стиль панелей: белая рамка 1px, белый текст, тёмно-синий фон, появляются снизу с fade

## Переводы

Реализовать через JS: все текстовые элементы имеют атрибут `data-i18n="ключ"`. JS-объект содержит переводы для всех трёх языков. При клике на кнопку языка — проходишь по всем элементам с `data-i18n` и меняешь `textContent`.

Валюта переключается одновременно с языком:
- EN → `$` / `USD`
- RU → `₽` / `RUB`
- BY → `Br` / `BYN`

Цены в карточках обновляются автоматически через `data-price-usd`, `data-price-rub`, `data-price-byn`.

## Важные требования к коду

1. Весь JS — в отдельных файлах по функциям (pendulum.js, blueprint.js, scroll-lock.js, main.js)
2. CSS-переменные для всех цветов и размеров — в `:root`
3. Никаких inline-стилей кроме JS-анимаций
4. Семантический HTML: `<nav>`, `<main>`, `<section>`, `<footer>`, `<article>`
5. Все секции имеют `id` для якорной навигации: `#catalog`, `#about`, `#preorder`
6. Код должен работать без сервера — просто открытием `index.html` в браузере
7. Комментарии в коде на русском языке — кратко поясняй что делает каждый блок
